Why Cosine vs Euclidean Distance?

Cosine: Compares shape of pattern
‚Üí Scale-invariant (direction only)

Euclidean: Compares magnitude too
‚Üí Sensitive to absolute values

Converting Similarity to Distance

From cosine similarity to cosine distance metric

Cosine Similarity Range: [-1, 1]

‚Ä¢ cos(Œ∏) = 1 ‚Üí vectors point in same direction (identical patterns)
‚Ä¢ cos(Œ∏) = 0 ‚Üí vectors are orthogonal (unrelated patterns)
‚Ä¢ cos(Œ∏) = -1 ‚Üí vectors point in opposite directions (inverse patterns)
‚¨áÔ∏è

Cosine Distance Formula:

distance = 1 - cos(Œ∏)

Distance Range: [0, 2]

‚Ä¢ distance = 0 ‚Üí identical patterns (cos = 1)
‚Ä¢ distance = 1 ‚Üí orthogonal patterns (cos = 0)
‚Ä¢ distance = 2 ‚Üí opposite patterns (cos = -1)

Example: Chelsea ‚Üî Wolves: cos(Œ∏) = 0.995 ‚Üí distance = 0.005 (very similar patterns!)
Crystal Palace ‚Üî Bournemouth: cos(Œ∏) = 0.313 ‚Üí distance = 0.687 (very different patterns)
